# ai-sandobx

## 運用メモ

### デプロイ手順

1. `npm run build` でビルドを実行する
2. Docker イメージをビルドしてレジストリへプッシュする
3. Kubernetes マニフェストを適用する
4. データベースのマイグレーションを実行する
5. ヘルスチェックで正常起動を確認する

### 監視・アラート

- CPU 使用率が 80% を超えた場合はアラートを発報する
- メモリ使用量が 90% を超えた場合はアラートを発報する
- エラーレートが 1% を超えた場合はオンコール担当へ通知する

### ロールバック手順

障害発生時は Blue-Green デプロイを利用して旧バージョンへ切り戻す。  
切り戻し後、必ずインシデントレポートを作成してください。

### よくある問題

| 症状 | 原因 | 対応 |
|------|------|------|
| DB 接続エラー | コネクションプール枯渇 | クエリキューイングを有効化する |
| Safari でページが壊れる | async generator 非対応 | Babel トランスパイラを確認する |
| レスポンスが遅い | N+1 クエリ | クエリログを確認し最適化する |

### 連絡先

障害・緊急時の連絡先は CLAUDE.md のチーム連絡先を参照してください。
